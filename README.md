# Screen Pet 🐱 - 화면 펫 확장프로그램

브라우저 화면 아래쪽을 뛰어다니는 귀여운 고양이 친구!

## 설치 방법

1. 이 폴더 전체를 압축 해제 (이미 풀려있다면 OK)
2. 크롬에서 주소창에 `chrome://extensions` 입력
3. 우측 상단 **"개발자 모드"** 토글 ON
4. 좌측 상단 **"압축해제된 확장 프로그램을 로드합니다"** 클릭
5. 이 `screen-pet` 폴더 선택
6. 아무 웹페이지나 열면 펫이 나타남! (이미 열려있던 탭은 새로고침 필요)

## 조작법

| 동작 | 결과 |
|---|---|
| **드래그** | 펫을 집어서 원하는 위치로 이동 |
| **더블클릭** | 펫이 점프하며 반가워함 |
| **우클릭** | 펫 숨기기 / 다시 보이기 |
| **마우스 커서를 하단에 두기** | 펫이 커서를 쫓아옴 |

## 동작

- 펫은 자동으로 **걷기 / 뛰기 / 점프 / 앉기**를 랜덤하게 반복
- 커서가 하단 근처에서 움직이면 쫓아옴
- 드래그해서 놓으면 중력 적용되어 떨어짐
- 꼬리 흔들기, 눈 깜빡이기, 걸을 때 다리 움직임 등 애니메이션

## 커스터마이징

- **특정 사이트에서 안 보이게 하려면**: `manifest.json`의 `matches`를 수정
- **색깔 바꾸기**: `content.js`의 SVG 안의 `fill` 색상 변경
  - 몸통: `#e89072` (주황), `#c8663d` (어두운 주황)
- **크기 변경**: `styles.css`의 `#screen-pet`의 `width/height`와 `content.js`의 `PET_W/PET_H` 동시 변경
- **속도 조정**: `content.js` 안에서 `vx = Math.sign(dx) * 숫자` 부분 수정
